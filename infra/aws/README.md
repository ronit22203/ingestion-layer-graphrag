# Deploying the GraphRag Ingestion Layer on AWS

## Complete end to end implementation using AWS services, including:
- **AWS Lambda** for serverless execution of the ingestion pipeline.
- **Amazon S3** for scalable storage of raw PDFs and intermediate outputs.
- **Amazon DynamoDB** for metadata management and tracking document processing status.
- **AWS EC2** for GPU-accelerated processing tasks, such as OCR and embedding generation.
